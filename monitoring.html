SYSTEM OVERVIEW
ISI Terminal v6.0 ek institutional-grade trading journal aur execution discipline platform hai. Yeh system retail traders ko institutional-level structure provide karta hai — jaise hedge funds aur prop desks use karte hain.

Har trade ke pehle mandatory pre-entry analysis hoti hai — HTF/LTF bias confirm hone ke baad hi terminal access milti hai.
DESIGN PHILOSOPHY
System is principle pe design hua hai: "Process over outcome."

Trader ka focus P/L pe nahi, execution quality pe hona chahiye. Agar process sahi hai — results automatically follow karte hain.

Har session mein trader ko apna HTF structure, LTF confirmation, SMC confluence aur psychological state document karna padta hai — tabhi execute kar sakta hai.
HOW IT WORKS
01 Pre-Entry Analysis — HTF/LTF bias engine, SMC flags, readiness score
02 Permission Matrix — 8-star execution flow, conflict detection
03 Trade Journal — Psychology, violations, scale, screenshot
04 AI Review — Gemini/Groq auto post-trade coaching
05 Monitoring — Calendar, equity curve, deep-dive analytics
WHY IT WAS BUILT
Most retail traders fail not because they lack strategy — but because they lack discipline infrastructure.

ISI Terminal fills this gap. It forces the trader to slow down, document every decision, and review every outcome — creating a feedback loop that compounds improvement over time.

"Institutional edge is not a strategy. It is a system."
